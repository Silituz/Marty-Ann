# Marty-Ann

A friendly birthday surprise page in English for Marty-Ann.

The design uses orange, comic-book energy, and web-slinger-inspired motion without using any official character images, logos, or romantic messaging.

## Open

Open `index.html` in a browser.

## Music

Add the birthday song later as:

```text
assets/birthday-song.mp3
```

The music button is already wired for that file.

## Photos

Add future pictures to the `assets` folder with these names:

```text
photo-01.jpg
photo-02.jpg
photo-03.jpg
photo-04.jpg
photo-05.jpg
photo-06.jpg
photo-07.jpg
photo-08.jpg
photo-09.jpg
```

The gallery and photo preview placeholders are already prepared for them.

## Text

All visible text is in English and intentionally written as a warm friendship birthday message, not a love message.
